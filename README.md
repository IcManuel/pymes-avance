# DynamoPymes — Tablero de avance de migración

Página estática autocontenida (`index.html`) que muestra el avance de la migración del ERP
DynamoPymes de JSF/PrimeFaces a React. Inventario de las pantallas `.xhtml` del legacy agrupadas
por módulo/tipo, con estado (migrada / pendiente / sin uso / fragmento), filtros y progreso.

> **Espejo público.** La fuente de verdad es `docs/avance-migracion.html` del repositorio privado
> `icreativateam/pymes-migracion`. Al terminar una pantalla se actualiza allí y se re-copia aquí
> (Regla de Oro). No contiene datos de clientes ni credenciales: solo nombres de pantallas,
> módulos y conteos.

Publicado con GitHub Pages.
